# Azkar TV Display Rev07.04

Hybrid Android TV / mobile WebView application for two master content types: Dua and Azkar.

## Rev07.04 Implementation

- Audio and Adhan remain excluded as instructed. Existing prayer alert chime has been disabled.
- Content library expanded to 272 total records, including 184 Dua and 88 Azkar records.
- Content index regenerated for rotation modes, tags, categories, authenticity status, and long-text records.
- Added local reading statistics, recent/history overlay, and most-read overlay.
- Added Arabic + Transliteration reading mode in addition to full, Arabic + Translation, and Arabic only modes.
- Added optional taxonomy fields: section, group, topic, occasion, time_of_day, virtue, difficulty, length, and language.
- Continued support for manual city selection, search overlay, favorites, swipe navigation, long-press favorite, and TV immersive mode.

## Current Content Governance

Records with exact Quran or hadith numbering are marked authentic or has_reference. Records requiring final scholarly numbering or grading remain marked needs_review. Manual verified tajweed_html is still not claimed as complete.
